MedHack: Frontiers 2026 – Data Folder
=====================================

All datasets used for our MedHack: Frontiers 2026 Small Tract (Coding) solution should be imported to this folder.

**Important: Data access & disclosure**
- These data files should come from the MedHack: Frontiers 2026 competition.
- The data is **not allowed to be publicly disclosed, redistributed, or uploaded** outside the competition environment.
- Access is restricted to registered participants of **MedHack: Frontiers 2026** (see the official site: [MedHack: Frontiers](https://www.med-hack.com/)).
- If you are viewing this repository outside the official competition context, you should **not** expect the raw data to be included. Only participants with valid access may download or use the original data.

**Files to be included in this folder**
- `train_data.csv` – Main training tabular dataset with patients' viyals and ground-truth labels.
- `test_data.csv` – Test tabular dataset with the same feature schema as `train_data.csv`.
- `holdout_data.csv` – Holdout set used for competition, without ground-truth labels.
- `patients.csv` – Static patient-level information.

Please keep all original competition data inside this `data` folder
